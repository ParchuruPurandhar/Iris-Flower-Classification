# 🌸 Iris Flower Classification

## Project Overview

This project predicts the species of an Iris flower using Machine Learning based on four flower measurements:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

The model classifies flowers into:

* Iris Setosa
* Iris Versicolor
* Iris Virginica
